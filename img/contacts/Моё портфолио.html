Клинцов Александр

Здравствуйте, меня зовут Константин Манихин. Я занимаюсь HTML-версткой сайтов, в дальнейшем планирую основательно изучать JavaScript, популярные библиотеки и фреймворки для него, такие как: jQuery, Ember.js, Angular.JS, React.JS.

В своей работе я использую сборщик проектов Gulp для автоматизации трудоёмких задач, повторяющихся в процессе работы. Это значительно ускоряет процесс разработки. Прописываю стили в препроцессоре sass.

Для адаптивноcти сайта применяю фреймворк Bootstrap и css медиа запросы для точного позиционирования элементов сайта на разных размерах экрана. Добавляю иконки с помощью иконочного шрифта Font Awesome, если это не противоречит дизайну макета, работаю с svg форматом. Использую различные fron-end плагины.
